3 Where is the Collection?

If you obtained the server and client by installing a ".plt" file, then the "handin-server" and "handin-client" directories might be in your PLT addon space. Start Racket, and enter
  (collection-path "handin-server")
  (collection-path "handin-client")
to find out where these collections are.
